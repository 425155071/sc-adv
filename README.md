## 安装设置

### 安装依赖
``` bash
$ yarn install
```

### 在本地运行 （localhost:8080）
``` bash
$ yarn dev
```

### 生成运行于生产环境的页面 & 运行
``` bash
$ yarn build
$ yarn start
```

### 生成可在Github Page浏览的静态页面（/dist）
``` bash
$ yarn generate
```
