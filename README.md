## 安装设置

### 安装依赖
``` bash
$ yarn install
```

### 在本地运行 （localhost:8080）
``` bash
$ yarn dev
```

### 生成运行于生产环境的页面 & 运行
``` bash
$ yarn build
$ yarn start
```

### 生成&直接部署/dist到Github Pages（不需切换分支，可直接在master分支里操作）
``` bash
$ yarn generate:gh-pages
$ yarn deploy
```
